## Overview
Remindo is your personal life manager. Track bills, birthdays, routines, job applications, and important reminders — all in one place. Built with a stunning glassmorphism design, it offers a deeply immersive and premium experience in both Light and Dark modes.

<p align="center">
  <img src="www/assets/Preview.png" width="800" alt="Remindo App Preview"/>
</p>

## ✨ Features
- **Smart Tracking:** Manage Job Applications, Bills, Birthdays, Medicines, Wishlists, and more.
- **Premium UI:** Designed with a floating modern pill-navigation, blurred bottom sheets, and custom UI components beautifully balanced for maximum readability and visual impact.
- **Modern Calendar:** Integrated with a custom-themed `flatpickr` for a sleek, premium date-picking experience across all devices.
- **Light & Dark Mode Adaptive:** Dynamically responds to your system theme, delivering perfect contrast and an aesthetic appeal without missing a beat.
- **Native App Feel:** Layouts fit cleanly on mobile screens with responsive design, ensuring a seamless user experience.

## 🚀 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Libraries:** Flatpickr (Datepicker)
- **Backend/Platform:** Firebase & Capacitor
